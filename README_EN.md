## 🎯 About

**PetFinder** is a web application designed to facilitate responsible dog and cat adoption. The system connects registered NGOs with potential adopters, centralizing information and reducing animal abandonment in the Salvador metropolitan area.

&gt; Developed with focus on UX/UI, accessibility, and performance.

---

## ✨ Features

- 🔍 **Advanced search** for pets by city, species, size, and age
- 🏠 **NGO registration** with admin dashboard
- ❤️ **Favorites system** to save pets of interest
- 🌙 **Light/dark theme** with local persistence
- 📱 **Responsive design** for mobile and desktop
- ⚡ **Fast loading** with Vite and lazy loading

## 🚀 Installation

### Prerequisites
- Node.js 18+
- npm or yarn

### Step by step

```bash
# Clone the repository
git clone https://github.com/Rsenju/projeto_petfinder.git

# Navigate to folder
cd petfinder-react

# Install dependencies
npm install

# Start development server
npm run dev
```

## 📁 Project Structure

| `src/components/features/` | Feature-specific components |
