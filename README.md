## 🎯 Sobre

O **PetFinder** é uma aplicação web desenvolvida para facilitar a adoção responsável de cães e gatos. O sistema conecta ONGs cadastradas com potenciais adotantes, centralizando informações e reduzindo o abandono de animais na região metropolitana de Salvador.

> Projeto desenvolvido com foco em UX/UI, acessibilidade e performance.

---

## ✨ Funcionalidades

- 🔍 **Busca avançada** de pets por cidade, espécie, porte e idade
- 🏠 **Cadastro de ONGs** com painel administrativo
- ❤️ **Sistema de favoritos** para salvar pets de interesse
- 🌙 **Tema claro/escuro** com persistência local
- 📱 **Design responsivo** para mobile e desktop
- ⚡ **Carregamento rápido** com Vite e lazy loading

## 🚀 Instalação

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/Rsenju/projeto_petfinder.git

# Entre na pasta
cd petfinder-react

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
