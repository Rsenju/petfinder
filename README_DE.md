## 🎯 Über

**PetFinder** ist eine Webanwendung zur Erleichterung der verantwortungsvollen Adoption von Hunden und Katzen. Das System verbindet registrierte NGOs mit potenziellen Adoptanten, zentralisiert Informationen und reduziert Tierverlassenheit in der Metropolregion Salvador.

> Entwickelt mit Fokus auf UX/UI, Barrierefreiheit und Performance.

---

## ✨ Funktionen

- 🔍 **Erweiterte Suche** nach Tieren nach Stadt, Art, Größe und Alter
- 🏠 **NGO-Registrierung** mit Admin-Dashboard
- ❤️ **Favoritensystem** zum Speichern interessanter Tiere
- 🌙 **Helles/dunkles Thema** mit lokaler Speicherung
- 📱 **Responsives Design** für Mobile und Desktop
- ⚡ **Schnelles Laden** mit Vite und Lazy Loading

## 🚀 Installation

### Voraussetzungen
- Node.js 18+
- npm oder yarn

### Schritt für Schritt

```bash
# Repository klonen
git clone https://github.com/Rsenju/projeto_petfinder.git

# In Ordner wechseln
cd petfinder-react

# Abhängigkeiten installieren
npm install
```

## 📁 Projektstruktur

| Verzeichnis | Beschreibung |
|-------------|--------------|
| `src/components/features/` | Funktionsspezifische Komponenten |
| `src/components/layout/` | Seitenstruktur (Header, Footer) |
| `src/components/ui/` | Wiederverwendbare Elemente (Buttons, Inputs) |

| `src/pages/` | Anwendungsseiten |
